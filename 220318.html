git bash : git-hub와 프로젝트를 연동할때 사용하는 작은 command도구, linux명령체계와 유사 윈도우에서 linux기본 명령을 연습할수 있다=Linux Bourne Shell 윈도우cmd : 윈도우에서 마우스를 사용하지 않고 키보드로 명령을 입력하여 어떤 기능을 수행할때 사용하는 도구 Linux Command Shell : linux 운영체제에서 키보드로 명령을 입력하여 어떤 기능을 수행할때 사용하는 명령입력 창(window) 깃헙 처음 연동 git config --global user.name git config --global user.email $ : prompt : 명령을 내리면(명령입력후enter) 수행할 준비가 되었다 라는 표식 clear : 현재 화면의 메시지를 지우는 명령어 github.com에 repository 생성 git init : git local 저장소 설정(초기화,신청) git remote add... git add . : 현재 폴더의 모든 파일, 모든 폴더( . )를 압축하여 .git 폴더에 저장 요청 git commit -m first git push -u origin master rm -rf .git : local repository 삭제(복구불가)
